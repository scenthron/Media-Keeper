# Media Keeper 📂✨

[![Downloads](https://img.shields.io/github/downloads/scenthron/Media-Keeper/total?style=for-the-badge&color=3b82f6)](https://github.com/scenthron/Media-Keeper/releases)

*🇷🇺 Русский | [🇬🇧 English](README.md)*

**Утилитарный офлайн-комбайн для удобной сортировки, глубокого анализа диска, безопасного удаления дубликатов и быстрого редактирования медиафайлов.**

Media Keeper — это мощное и полностью конфиденциальное приложение для организации, очистки и обработки вашего медиаархива. Программа объединяет в себе все необходимые инструменты для работы с изображениями, видео и аудио в повседневной жизни, сохраняя интерфейс чистым, интуитивным и не перегруженным лишними настройками.

---

## 💎 Главные особенности

*   **🛡️ 100% Конфиденциальность и Безопасность:**
    *   Работает полностью **без подключения к интернету**. Все операции выполняются исключительно локально.
    *   Приложение **не собирает**, **не передаёт** и **не отправляет** никакие файлы, метаданные или логи третьим лицам.
    *   В программе **отсутствуют** скрытые облачные сервисы и системы искусственного интеллекта.
*   **⚡ Удобный и живой интерфейс (дизайн-система TITAN):**
    *   Современная тёмная тема с адаптивной версткой.
    *   Стилистика некоторых элементов отсылает к культовой пиксельной игре **Graveyard Keeper**.
    *   Поддержка **Drag & Drop** для быстрого анализа папок и настройки параметров.
    *   Удобная и настраиваемая система горячих клавиш.
*   **🔌 Портативность (Portable):**
    *   Программа не требует установки в операционную систему и может запускаться с любого носителя (флешка, внешний диск).

## 🛠️ Основные модули программы

### 1. 📂 Сортировщик Медиафайлов (Sorter)
Предназначен для удобного интерактивного просмотра медиа и других типов файлов, а также их быстрой сортировки по целевым папкам в один клик.

*   **Воркфлоу:** Выбираете папку-источник (Inbox) ➡️ Указываете целевую папку или создаете новые каталоги прямо в процессе ➡️ Задаете папку для мусора ➡️ Сортируете файлы кликом по дереву каталогов или горячей клавишей.
*   **Ключевые возможности:**
    *   **3 режима просмотра:** Плитка (по умолчанию), Список и Индивидуальный просмотр.
    *   **Быстрый просмотр:** Мгновенный предпросмотр картинок, аудио и видео при наведении или клике.
    *   **Умные фильтры:** Фильтрация по типам файлов на входе и опция «Учитывать вложенные файлы» (рекурсивный обход подкаталогов).
    *   **Панель аффиксов (тегов):** Быстрое добавление префиксов, суффиксов, тегов даты/времени к группе файлов (до 40 штук) с предварительной транзакционной симуляцией (Dry Run) для защиты от конфликтов имен.
    *   **Безопасный откат (Undo):** Отмена последнего перемещения/переименования по нажатию `Ctrl + Z`.
    *   **Масштабирование:** Увеличение/уменьшение контента комбинацией `Ctrl + скролл мыши` в режимах плитки и списка.
    *   **Кастомизация дерева:** Фокусировка на выбранной ветке дерева, кастомные иконки папок из встроенного набора.

### 2. 📊 Анализатор пространства диска (Analyzer)
Визуализирует использование дискового пространства папок и файлов, помогая быстро находить самые «тяжелые» каталоги и файлы.

*   **Воркфлоу:** Выбираете папку для сканирования ➡️ Автоматически строится диаграмма ➡️ Навигируетесь по секторам для детального анализа.
*   **Ключевые возможности:**
    *   **Интерактивные диаграммы:** Поддерживаются два режима — плоская плиточная визуализация (**TreeMap** по алгоритму Squarified) и круговая (**Sunburst**).
    *   **Быстрое управление:** Изменение глубины вложенности с помощью комбобокса или скролла мыши при наведении на диаграмму. Кнопка случайной смены цветовой палитры.
    *   **Центральный оверлей:** Навигационные кнопки (Назад, На уровень вверх) и отображение размера выбранной папки прямо по центру сцены.
    *   **Двойная детализация:** Правая таблица группирует файлы по расширениям, а нижняя показывает конкретные файлы в выбранной категории с возможностью группировки по каталогам.
    *   **Анализ папки из контекстного меню:** Быстрый переход к детальному анализу родительской папки любого файла.
    *   **Быстрый поиск:** Умный поиск по именам файлов во всей просканированной структуре.

![Модуль Анализатора диска](images/analyzer_module.png)

### 3. 🔍 Поиск и удаление дубликатов (Cleaner)
Помогает очистить диск от повторяющихся файлов на основе расчета хэш-сумм файлов (даже если файлы переименованы).

*   **Воркфлоу:** Добавляете папки-источники ➡️ Настраиваете типы сканируемых файлов ➡️ Запускаете сканирование ➡️ Применяете автофильтры или вручную выделяете дубликаты ➡️ Удаляете или переносите их в папку для анализа.
*   **Ключевые возможности:**
    *   **🛡️ Железобетонное правило безопасности (N-1):** Программа физически не позволит отметить и удалить абсолютно все файлы из группы дубликатов. Как минимум один оригинальный файл всегда останется целым («выживший»).
    *   **Защита папок и «Поиск по эталону»:** Вы можете назначить одну из папок эталонной. Дубликаты будут искаться только для файлов из этой папки, при этом сами эталонные файлы защищены от удаления.
    *   **Сохранение структуры путей:** При переносе дубликатов в архивную папку сохраняется исходная структура вложенности папок для возможности безопасного возврата.
    *   **Поиск битых файлов и пустых папок:** Нахождение файлов размером 0 байт и пустых каталогов для последующей очистки.
    *   **Умная сортировка результатов:** Упорядочивание групп дубликатов по степени их обработанности.

### 4. 🎬 Видеоредактор и Конвертеры (Editor)
Объединяет возможности быстрой lossless-обрезки, кадрирования видео и пакетной смены форматов медиафайлов.

*   **Видеоредактор:** Обработка видео с мгновенным предпросмотром.
    *   **Трансформация:** Поворот, зеркальное отражение, кадрирование (Crop).
    *   **Цветокоррекция:** Настройка яркости, контрастности, насыщенности и гаммы.
    *   **Наложения:** Добавление размытия на выбранную область (Blur), цветной заливки или логотипа-изображения.
    *   **Маркеры фрагментов:** Быстрая нарезка видео на сегменты или вырезание ненужных частей с возможностью экспорта в отдельные файлы.
*   **Пакетные Конвертеры (фоновый режим):**
    *   **Видеоконвертер:** Смена кодеков (H.264 / H.265), изменение разрешения и битрейта (CRF) со сжатием видео под конкретный лимит размера (для мессенджеров).
    *   **Аудиоконвертер:** Сжатие аудио, конвертация в MP3, AAC, FLAC с поддержкой CBR/VBR, извлечение звуковых дорожек из видео.
    *   **Конвертер изображений:** Пакетное изменение размеров, смена форматов (PNG, JPG, WebP) и качества.

## 🚀 Установка и запуск (для разработчиков)

### Системные требования
*   Python 3.10 или новее
*   Операционная система: Windows 10/11 (для сборки и полноценной работы)
*   **FFmpeg:** Необходим для работы видеоредактора и конвертеров. Программа автоматически предложит скачать и установить его в свою системную директорию `.mediakeeper/bin/` при первом запуске.

### Шаги по установке
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/scenthron/Media-Keeper.git
   cd Media-Keeper
   ```
2. Создайте и активируйте виртуальное окружение (рекомендуется):
   ```bash
   # Создание виртуального окружения
   python -m venv venv

   # Активация окружения (Windows)
   venv\Scripts\activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение через скрипт-менеджер (он автоматически прогонит автотесты перед запуском):
   ```bash
   python run.py
   ```

## 📦 Сборка в исполняемый файл (.exe)

Сборка осуществляется с помощью кастомного скрипта `build.py`, который оптимизирует размер EXE, автоматически импортирует необходимые SVG-модули Qt6 и включает папки ресурсов:

```bash
python build.py
```
После завершения сборки готовый файл `.exe` появится в корневой папке `dist/`.

## 🤖 Автоматическая сборка в облаке (CI/CD)

В репозитории настроен рабочий процесс **GitHub Actions** (файл [.github/workflows/build.yml](.github/workflows/build.yml)):
*   При каждом `git push` в ветку `master` запускается тестовая сборка, артефакт которой можно скачать во вкладке **Actions** (хранится 30 дней).
*   При публикации релиза (Release) на GitHub робот автоматически компилирует чистую Windows-версию приложения, запаковывает её в ZIP и прикрепляет к релизу в виде готового дистрибутива.

## 📜 История создания
Media Keeper родился из множества разрозненных скриптов для автоматизации рутины, которые автор создавал под собственные нужды. Желание объединить этот функционал в едином приложении долгое время оставалось нереализованным, так как автор не является профессиональным программистом.

Появление современных нейросетей (в частности, ИИ-ассистента **Google Antigravity**) открыло новые возможности и позволило воплотить все накопленные идеи в едином продукте.

Название программы вдохновлено любовью автора к пиксельной игре **Graveyard Keeper**, элементы дизайна которой вы можете встретить в интерфейсе.
