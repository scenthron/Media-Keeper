# Media Keeper — General Information

**Media Keeper** is a powerful offline tool for organizing, analyzing, cleaning, and processing your media archive.

The program provides a wide range of features for convenient and efficient media content management, allowing you to quickly bring order to your file collection.

### 💎 Core Concept

The main goal of developing Media Keeper was and remains to create a single, convenient space where all the necessary tools for working with media files in everyday life are collected. At the same time, the interface remains clean and is not overloaded with redundant settings, ensuring intuitive and comfortable use.

### 🛡️ 100% Privacy and Security
- The program works completely **without an internet connection**.
- All operations are performed exclusively locally on your device.
- The application **does not collect**, **does not transfer**, and **does not send** any files, metadata, or other information to third parties.
- Media Keeper is completely free and contains no ads.
- There are **no** Artificial Intelligence systems or cloud services in the program.
- A multi-level data protection system has been implemented when processing files.

### 🛠️ Technical Features
- **⚠️ Important recommendation:** For correct operation of the program, it is recommended to completely extract it from the archive before launching. Launching directly from the archive may lead to unstable operation and potential loss of settings.
- **Drag & Drop Support:** The program supports Drag & Drop technology. To analyze folders and configure parameters, simply drag the required folders into the corresponding functional areas of the interface. These zones are usually indicated by a dashed line or labeled.
- **Hotkeys:** Media Keeper supports a convenient hotkey system. All key settings are stored in the program's system folder inside the `.mediakeeper` directory. To preserve settings when moving or copying the program, be sure to transfer this folder along with the main application files.
- **Using FFmpeg:** To process media files, the program uses the free, open-source **FFmpeg** library. It is not pre-packaged and is downloaded automatically at the first launch (or manually by following the instructions). The library is saved to the program's system folder. When copying or moving the application, it is recommended to preserve the entire system folder to keep settings and required components.
- **Portable Version:** The program is fully portable — it does not require installation in the operating system and can be run from any media (USB flash drive, external drive, etc.).

### 📜 History of Creation
Media Keeper was born out of many separate scripts and small tools for automation and media file management that the author created for his own needs.

Over time, there was a desire to combine all this functionality in a single, convenient, and intuitive application. Since the author is not a professional programmer and has no professional IT education, implementing such a task remained practically impossible for a long time.

The emergence of modern neural networks opened up new opportunities and allowed to merge all concepts and ideas into a single product. That is how the Media Keeper program appeared.

The name of the program expresses the author's deep love for the cult pixel game **Graveyard Keeper**. That is why some elements of the interface are styled after this game.
