# 📂 Media Sorter

**The primary and highly feature-rich section of the program. It is designed for convenient interactive viewing of media and other file types, and quick sorting of files into target folders in a single click.**

### 🚀 Workflow
1.  **Choose Inbox:** Select the source folder (Inbox) in the left panel or by dragging and dropping a folder onto the canvas of the section.
2.  **Configure Targets:** Specify a target folder that already contains sorted subfolders, or create a new section and populate it during the file organization process.
3.  **Trash Folder:** Set a target folder for junk files. You can quickly filter out unwanted files into a separate directory to decide on their fate later.
4.  **Sort:** Choose a file or a group of files by a key parameter and quickly move the selected files to the corresponding folder simply by clicking on it in the directory tree.

### ⚙️ Key Features
*   **3 View Modes:** You can view files in your preferred layout: Grid (default), List, or Individual View.
*   **Quick Preview:** Instant preview of images, audio, and video on hover or click. Files supporting quick preview have a green label with their extension. The quick preview window is configured within the session.
*   **Inbox Filter:** By clicking the funnel icon on the top panel, you can choose which file types you want to sort or which types you want to exclude from the current list. You can also enable the "Include Subfolders" option, allowing the program to scan files not only in the main folder but also inside all nested subdirectories.
*   **File Sorting:** You can configure various sorting modes for the displayed files: "By type", "By alphabet", and also enable sorting by groups.
*   **Affix Panel:** The affix panel allows you to configure quick tags. By selecting files and clicking on a tag, you can rename the files according to the created rule. For example, you can append prefixes, date tags, or job-related notes. An individual rename button is also available for manually renaming files one by one.
*   **Safe Undo:** Cancel the last move or rename action by pressing `Ctrl+Z`.
*   **Advanced Media Player:** A convenient media player is available for viewing media files, featuring playback speed control, volume adjustment, and looping. Zooming is supported for images.
*   **Zooming:** Supports zooming in/out of the displayed content in Grid and List modes using the Ctrl + Mouse Scroll combination.

### 🌳 Directory Tree Features
*   **Navigation:** For complex tree structures, you can "focus" on a specific folder, temporarily making it the root folder, with the option to return to the full view.
*   **Custom Folder Icons:** Click the gear icon ⚙️ next to the folder you want to customize. This opens a settings window where you can select an icon from a wide list. Settings are saved in the program's system folder in the `.mediakeeper` directory.
*   **Assign Quick Target Folder:** You can set a folder as the "Quick Target" by right-clicking the target folder and selecting the corresponding option. This enables moving files instantly using a hotkey on your keyboard (Space by default).
*   **Add External Directories:** You can drag and drop any external folder from the OS into the directory tree to add a temporary folder with additional sorting categories.
*   **Folder Automation (Rename on Move):** You can set up automatic file renaming when moving files into a specific folder. To do this, click the gear icon ⚙️ on the desired folder and configure the renaming template. For example, you can set a rule that prepends the prefix "Vacation_2026_" to all files placed in the folder while keeping the original file name. This makes it easy to find files in the future without spending time renaming each file manually.

### ⌨️ Hotkeys
*   `Arrow keys` — navigate to the previous/next file.
*   `Space` — play/pause media files or quickly move the file to the assigned quick target directory.
*   `Del` — send selected files to the "Trash Folder".
*   `Ctrl+Z` — undo the last action.
