# 📊 Анализатор пространства диска 

**Раздел визуализирует использование дискового пространства, папок и файлов. Помогает при помощи диаграмм TreeMap и Sunburst быстро находить «тяжелые» каталоги и файлы.**

### 🚀 Порядок работы (Воркфлоу)
1.  **Выбор директории:** Перетащите папку прямо на область диаграммы или выберите путь в верхней строке.
2.  **Сканирование:** Автоматически произойдет сканирование выбранного диска или папки и по завершении сканирования будет показана диаграмма. Настройка уровня вложенности в верхней панели ограничивает отображение вложенности папок в каталогах (можно менять скроллом мыши при наведении).
3. **Дизайн диаграммы:** вы можете менять дизайн диаграммы, выбирая между круговым и плиточным вариантами. Также доступна кнопка для случайной смены цветовой палитры.
4.  **Анализ:** Кликайте по секторам оранжевого цвета интерактивной диаграммы для перехода внутрь папок и более детального анализа. Кнопки навигации помогут вернуться назад или перейти на папку выше.

### ⚙️ Ключевые возможности
*   **Интерактивные диаграммы:** Наглядная визуализация структуры каталогов.
*   **Оверлей в центре:** Показывает размер и имя выбранной папки, содержит кнопки навигации (Назад, На уровень вверх).
*   **Детализация категорий:** Правая таблица группирует файлы по расширениям, найденным в сканируемой директории. Показывает общее количество и размер всех файлов по их расширению.
*   **Детализация расширений:** В нижней таблице показывается детализация по файлам в определенной категории расширений, выбранной в правой таблице. Файлы можно сгруппировать по каталогам хранения.
*   **Целевая папка для перемещения:** В заголовке нижней таблицы можно задать папку для переноса выбранных в таблице файлов с целью их последующего анализа.
*   **Перемещения в указанные каталоги:** Через контекстное меню диаграммы или через таблицу файлов можно перемещать файлы в указанную пользователем директорию.
*   **Контекстное меню:** Доступно контекстное меню с различными удобными функциями по ПКМ на диаграмме и по файлам в таблице.
*   **Пакетные операции:** Выделение файлов галочками для быстрого удаления или переноса в другую папку.
*   **Поиск:** Доступен быстрый «умный» поиск по названиям файлов в просканированной директории.
