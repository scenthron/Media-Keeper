# 🔍 Поиск и удаление дубликатов

Мощный инструмент для очистки диска от повторяющихся файлов на основе расчета хэш-сумм (даже если файлы переименованы).

---

### 🚀 Порядок работы (Воркфлоу)
1.  **Выбор источников:** Добавьте папки для поиска дубликатов в раздел «Источники поиска».
2.  **Настройка типов:** Выберите сканируемые типы файлов в матричном фильтре. По умолчанию сканируются только медиафайлы.
3.  **Сканирование:** Нажмите **Сканировать** для поиска дубликатов.
4.  **Очистка:** Примените быстрые фильтры (например, «Выделить все с самым коротким именем»). Выделенные файлы можно как удалить, так и перенести в отдельную папку, предварительно настроив её.
5.  **Фильтры:** Итоговые результаты найденных дубликатов можно отсортировать при помощи фильтра в заголовке таблицы.
6.  **Сортировка:** При обработке результатов доступна удобная сортировка групп дубликатов. По нажатию на кнопку со стрелочками «вверх-вниз» в заголовке таблицы группы будут упорядочены по степени обработанности. В самом низу списка окажутся группы, в которых корректно выделены все «лишние» дубликаты (то есть остался ровно один исходный файл). В середине списка расположатся группы, где не выделен ни один файл. В самом верху будут находиться группы, где часть дубликатов уже выделена, но ещё не все лишние файлы отмечены. Для удобства пользователя сортировка происходит только в момент нажатия кнопки.

### 🛡️ Железобетонное правило безопасности (N-1)
При любых автоматических или ручных операциях программа **никогда** не позволит пометить и удалить все файлы из группы дубликатов. Как минимум один оригинальный файл всегда останется невыделенным («выживший»). Это правило действует в целях сохранности данных пользователя. В случае необходимости вы можете перейти в каталог с файлом и удалить его средствами операционной системы.

### ⚙️ Ключевые возможности
*   **Защита папок:** Возможность отметить папку как «Защищенную» (файлы из нее никогда не будут удалены).
*   **Поиск по эталону:** Будет искать дубли только тех файлов, которые хранятся в директории, отмеченной как «Эталон». Удобно, если вы хотите найти дубликаты только определенных файлов, не отвлекаясь на другие. Необходимо добавить несколько каталогов и один из них назначить «Эталонным». Файлы из категории «Эталонная» также защищены от удаления.
*   **Автоматическое разрешение коллизий:** Правила инкремента имен при переносах файлов с одинаковым названием.
*   **Сохранение оригинальных путей:** При переносе файлов-дубликатов в папку для временного анализа по умолчанию включена опция сохранения путей оригинальных файлов. Это позволит в случае необходимости переместить файлы обратно без изменения структуры их изначального нахождения.
*   **Поиск битых файлов и пустых папок:** При сканировании программа автоматически находит пустые файлы (0 байт) и папки, не содержащие файлов, которые вы можете удалить при необходимости. Однако помните, что файлы размером 0 байт могут использоваться как системные в некоторых программах и структурах. Поэтому используйте эти инструменты с особой внимательностью.
