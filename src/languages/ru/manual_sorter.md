# 📂 Сортировщик Медиафайлов

**Основной раздел программы с широкими возможностями. Предназначен для удобного интерактивного просмотра медиа и других типов файлов, а также их быстрой сортировки по целевым папкам в один клик.**

### 🚀 Порядок работы (Воркфлоу)
1.  **Выбор входящих:** Выберите папку-источник (Inbox) в левом меню или перетащив папку на холст раздела.
2.  **Настройка целей:** Укажите уже готовую папку с отсортированными файлами по папкам или создайте новый раздел и наполняйте его в процессе разбора файлов.
3.  **Папка для мусорных файлов:** Задайте папку для мусорных файлов. Вы сможете быстро отсортировать ненужные файлы в отдельный каталог, чтобы позже принять решение об их дальнейшей судьбе.
4.  **Сортировка:** Выбирайте файл или группу файлов по ключевому параметру и быстро переместите выбранные файлы в соответствующую папку, просто кликнув по ней в дереве каталогов.

### ⚙️ Ключевые возможности
*   **3 вида отображения файлов:** Вы можете просматривать файлы в удобном для вас виде: Плитка (по умолчанию), Список, Индивидуальный просмотр.
*   **Быстрый просмотр:** Мгновенный предпросмотр картинок, аудио и видео при наведении или клике. Файлы, поддерживающие быстрый просмотр, имеют зеленый лейбл с расширением. Окно быстрого просмотра настраивается в рамках сессии.
*   **Фильтр входящих файлов:** Нажав на кнопку с лейкой на верхней панели, вы сможете выбрать, какие типы файлов вы хотите сортировать или какие типы файлов не хотите видеть в итоговой выборке. Также в этом разделе можно включить функцию «Учитывать вложенные файлы», при которой программа будет сканировать файлы не только в указанной директории, но и во всех каталогах, находящихся в этой же папке включительно.
*   **Сортировка файлов:** Вы можете настроить различные режимы сортировки отображаемых файлов: «По типу», «По алфавиту», а также включить сортировку по группам.
*   **Панель аффиксов:** Панель аффиксов позволяет настроить быстрые метки. Выбрав файлы и нажав на метку, вы сможете изменить название файла в соответствии с созданным правилом. Например, добавить к файлам пометки, теги с датой или пометку о том, что файл относится к работе. Также доступна кнопка для индивидуального переименования каждого файла вручную.
*   **Безопасный откат (Undo):** Отмена последнего действия перемещения/переименования по нажатию `Ctrl+Z`.
*   **Удобный медиаплеер:** Вам доступен удобный проигрыватель медиафайлов с возможностью настройки скорости воспроизведения, громкости звука и повтора. Для изображений доступно масштабирование.
*   **Изменение масштаба:** Поддерживается увеличение и уменьшение отображаемого контента в режимах плитки и списка комбинацией Ctrl + скролл мыши.

### 🌳 Возможности дерева каталогов
*   **Навигация:** При сложной структуре дерева есть возможность «фокусироваться» на одной из папок, делая её на время главной, с возможностью возврата к общему виду.
*   **Добавление иконок к папкам:** Нажмите на кнопку ⚙️ у папки, для которой хотите задать иконку. Откроется окно настроек, где можно будет выбрать иконку из широкого списка. Настройки сохраняются в системную папку .mediakeeper.
*   **Назначение папки "Для быстрого перемещения":** Вы можете установить настройку, нажав ПКМ по целевой папке и выбрав соответствующий пункт в меню. Для этой папки станет возможным быстрое перемещение файлов по нажатию на горячую клавишу клавиатуры. По умолчанию — Пробел.
*   **Добавление внешних каталогов:** Вы можете перетащить дополнительный каталог из операционной системы в дерево каталогов, чтобы добавить временную папку с дополнительными разделами сортировки.
*   **Переименование файлов в целевой папке:** Вы можете настроить автоматическое переименование файлов при их перемещении в определенный каталог. Для этого на нужной папке нажмите кнопку с шестеренкой ⚙️ и настройте шаблон автоматического переименования. Например, вы можете настроить правило, при котором к имени всех помещаемых в папку файлов будет добавляться префикс «Отпуск_2026_» с сохранением оригинального имени. Так вы в будущем легко сможете находить нужные вам файлы, не тратя время на переименование каждого файла вручную.
  
### ⌨️ Горячие клавиши
*   `Стрелки на клавиатуре` — переход к предыдущему/следующему файлу.
*   `Space` — воспроизведение / пауза медиафайла или быстрое перемещение файла в установленную директорию.
*   `Del` — отправить выбранные файлы в «Корзину для мусорных файлов».
*   `Ctrl+Z` — откат последнего действия.
